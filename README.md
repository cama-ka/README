## Créé ton readme.
### Création d'un fichier simple au format .md.
 Suit les instructions, c'est facile !
 Le programme créer un fichier readme.md et effectue la mise en page avec les informations que l'utilisateur indique dans le terminal de commande.

### Pré-requis 
* Installer python3 : https://www.python.org/downloads/https://www.python.org/downloads/

### Programme
 Via github : tapez dans votre console : git clone https://github.com/cama-ka/readme.git

###Créer l'environnement virtuel 
 - Tapez dans votre console : python3 -m venv env
#### Programme
 Via la console, tapez : python scrap_one.py
