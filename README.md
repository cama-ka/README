## Creer un readme
### Fichier format .md
 C'est facile !
 Le programme va créer pour l'utilisateur un fichier readme.md simplement en répondant aux questions demandés sur le terminal de commande.

#### Pré-requis 
* Installer python3

#### Télécharger le programme:
 Via github : tapez sur le terminal de commande : git clone https://github.com/cama-ka/readme.git

####  Créer l'environnement virtuel 
 - Tapez sur le terminal de commande : python3 -m venv env 

####  Lancer le programme
 Tapez sur le terminal de commande : python script.py
